YouTube Clone

A fully responsive front-end clone of YouTube built using HTML, CSS, and JavaScript.

Live Link

Project Overview

This project is a front-end replica of the YouTube user interface, developed using vanilla HTML, CSS, and JavaScript. The layout mimics the core YouTube homepage, including the sidebar, video cards, and responsive grid system.

Key elements like video thumbnails, titles, and navigation behavior are designed to resemble the real YouTube experience. The project emphasizes modern UI principles, responsiveness across devices, and clean design practices.

This project is a demonstration of my front-end skills in crafting intuitive interfaces from scratch, useful for building dashboard UIs, streaming sites, or video-based content platforms.
